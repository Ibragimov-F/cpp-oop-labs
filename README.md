# 📘 C++ Object-Oriented Programming Labs

This repository contains my laboratory assignments for the **Object-Oriented Programming (OOP)** course completed during the **2nd semester of the 1st academic year**.

All tasks are implemented in **C++** and demonstrate fundamental principles of object-oriented programming through practical examples and exercises which professor provided.

* **Language:** C++
